## ANIMATION AND INTERACTION GUIDELINES

- All sections use scroll-triggered animations — elements fade in and slide up as they enter the viewport
- The hero code window has a continuous typing animation
- The nav bar hero action button ("Get a Free Mockup →" or "Start Your Site →") pulses subtly or has a gentle glow on page load to draw attention, then settles into a static state
- The sticky nav should have a slight background blur/opacity change when the user scrolls past the hero
- Transitions between sections should feel smooth and continuous
- All animations CSS-based or lightweight JS — no heavy libraries
- Performance first — fast loading is non-negotiable since we're selling speed and SEO
